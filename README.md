## dashboard-prevencao-fraudes-powerbi# 

📊 Dashboard de Prevenção à Fraude e Análise de Clientes

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de monitorar indicadores relacionados à prevenção de fraudes, permitindo identificar cidades com maior risco, acompanhar métricas estratégicas e apoiar a tomada de decisão.

Este é um dos projetos desenvolvidos durante minha transição de carreira para a área de Dados.

---

## 🚀 Tecnologias utilizadas

- Power BI
- SQL
- DAX
- Modelagem de Dados

---

## 📈 Indicadores

- Total de Clientes
- Total de Fraudes
- Percentual de Fraudes
- Score Médio
- Saldo Total
- Plano de Ação Dinâmico

---

## ✨ Funcionalidades

- KPIs dinâmicos
- Botão para limpar filtros
- Atualização automática da data
- Mapa por cidade
- Plano de ação baseado no nível de risco
